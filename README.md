# 🎉 API REST - Gerenciamento de Eventos

API REST desenvolvida em Java com Spring Boot para gerenciamento de eventos. A aplicação permite realizar operações de cadastro, consulta, atualização e exclusão de eventos por meio de requisições HTTP.

---

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar o desenvolvimento de APIs REST utilizando Spring Boot, aplicando arquitetura em camadas, boas práticas de programação e integração com banco de dados.

---

## 🚀 Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- MySQL 
- Maven
- Swagger/OpenAPI (se utilizado)
- Postman
- Git
- GitHub
